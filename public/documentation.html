This is a web application for people interested in movies. It will feature a database of information inclding generes, directors etc. Users will also have the ability to sign up and post personal information along with lists of their favorite movies.
REQUEST URL HTTP METHOD
Get a list of all movies /movies GET
Get data(descr, genre, director,
imageURL, featured or no) by title		 /movies/[title] GET
Get data about genre by title /movies/genre/[title] GET
get data about director by name /movies/director/[name] GET
add user (regist) /users POST
users update their info /users[info]/[username] PUT
users add movie to fav list /users[info]/favorites POST
users remove movie from fav list /users[info]/favorites DELETE
users deregister /users DELETE
