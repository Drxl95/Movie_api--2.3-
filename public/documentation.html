This is a RESTful API that contains data about certain movies, their genres, and their directors. There is also a membership feature where people can create accounts, post personal info, and add movies to their favorites list. It was built with the MERN stack. All Features - Membership site (users can create and account for free) - Members can add or remove movies to or from their favorites list - When a movie is clicked, and image, description, genre, director, and a list of primary actors are listed. - There is also a button for director info (bio, birthyear etc.) and another button for genre info. - Members can deregister if they wish
